### OpenAi官方库支持

> 在openai官方库开发时传入baseurl和apukey即可。

以官网的`python`库为例：注意，需要传入/v1后缀。

```python
import openai
openai.api_base = "https://free.gpt.ge/v1"
```

或者通过环境变量设置，具体参考官网库文档介绍

```python
OPENAI_API_BASE=https://free.gpt.ge/v1
```
